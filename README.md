# 🦅 Meu Timão - Projeto Corinthians

Este é um projeto web dedicado ao **Sport Club Corinthians Paulista**. O site foi desenvolvido utilizando tecnologias fundamentais da web (HTML, CSS e JavaScript) para criar uma experiência interativa e informativa para os torcedores fiéis do clube.

---

## 🎯 Objetivo do Projeto

O principal objetivo deste projeto é celebrar a história, as conquistas e a paixão do Corinthians através de uma plataforma web moderna e responsiva. O site serve tanto como um portfólio de desenvolvimento frontend quanto como um espaço para que os torcedores possam interagir, conferir informações sobre o clube e consumir conteúdos multimídia de forma dinâmica.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído do zero, utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica de todo o conteúdo da página (textos, seções e incorporação de mídias).
* **CSS3:** Estilização personalizada utilizando uma paleta de cores baseada na identidade do clube (preto, branco e cinza), além de layouts responsivos para dispositivos móveis e desktops.
* **JavaScript (ES6+):** Implementação de interatividade, manipulação do DOM e lógica para as funcionalidades dinâmicas do site.

---

## ✨ Funcionalidades

* **Central de Mídia:** Incorporação de vídeos do YouTube através de `iframe` para que o usuário assista aos melhores momentos e cantos da torcida diretamente no site.
* **Design Responsivo:** Interface adaptável que funciona perfeitamente em celulares, tablets e computadores.
* **Galeria Interativa:** Exibição dos principais títulos e ídolos históricos do clube com efeitos visuais modernos.
* **Efeitos Dinâmicos:** Menu de navegação interativo e animações suaves ao rolar a página utilizando JavaScript.
